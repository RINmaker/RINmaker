# RINmaker

## Configure project:

```bash
cmake -S . -B your-build-dir
```

## App

#### Build

```bash
cmake --build your-build-dir --target RINmaker
```

#### Run

```bash
./your-build-dir/app/RINmaker -h
```

## Test

#### Build

```bash
cmake --build your-build-dir --target RINmaker_test
```

#### Run

```bash
./your-build-dir/test/RINmaker_test
```
